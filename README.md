# Vehicle Pre-Booking System

## Project Overview
The Vehicle Pre-Booking System is a web-based application designed to support
vehicle rentals, showroom pre-booking, and travel package reservations through
a unified platform.

## Documents Uploaded (PDF Format)
1. UML Diagrams
2. Source Code
3. Output Files
4. Project Report
5. Project Presentation (PPT converted to PDF)

## Folder Structure
1_UML_Diagrams – Contains all UML diagrams  
2_Source_Code – Contains complete source code documentation  
3_Output – Contains output screenshots and results  
4_Project_Report – Contains the final project report  
5_PPT – Contains the project presentation  

## Technologies Used
Frontend: HTML, CSS, JavaScript  
Backend: PHP / Node.js  
Database: MySQL  
AI: NLP-based recommendation system
